🌾 Khetseva – A Smart Harvest

Khetseva is an intelligent agriculture assistance platform designed to help farmers make data-driven decisions. The system integrates Machine Learning and Deep Learning models to provide crop recommendations, plant disease detection, and crop price prediction, all in a single unified platform.

The goal of Khetseva is to bridge the gap between technology and agriculture by offering accurate, real-time insights that improve productivity and profitability.

📌 Problem Statement

Farmers often struggle with:

Choosing the right crop based on soil and environmental conditions

Identifying plant diseases at early stages

Predicting market prices to maximize profit

Khetseva addresses these challenges using machine learning models trained on real-world agricultural datasets.

🛠️ Key Features
1️⃣ Crop Recommendation System

Takes inputs like Nitrogen (N), Phosphorus (P), Potassium (K), temperature, humidity, rainfall, soil pH, and soil type

Uses ensemble ML models such as Random Forest and Decision Tree

Recommends the most suitable crop along with fertilizer suggestions

2️⃣ Plant Disease Detection

Uses image-based classification

Implemented using EfficientNetV2 and CNN architectures

Trained on a dataset of 90,000+ healthy and diseased plant images

Achieved high accuracy through model comparison and evaluation

3️⃣ Crop Price Prediction

Predicts crop prices based on yield and historical market trends

Evaluated using metrics like Accuracy, Precision, Recall, and F1 Score

Helps farmers plan selling strategies effectively


📊 Model Performance

Overall System Accuracy: 97.3% (based on comparative analysis)

Disease Detection: High classification accuracy using EfficientNetV2

Crop Recommendation: Strong ensemble performance

Price Prediction: >93% prediction accuracy
